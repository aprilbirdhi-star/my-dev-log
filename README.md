# 💻 PHILOG.TODAY (개발자 이루다의 성장 기록 공간)

기록하며 성장하는 개발자 **이루다**의 포트폴리오 및 일상/기술 로그를 담은 정적 웹사이트입니다.

* **배포 주소**: [https://philog.today](https://philog.today)
* **원격 저장소**: [GitHub Repository](https://github.com/aprilbirdhi-star/philog.today)

---

## 🚀 주요 기능

1. **AI 기반 일상 감정 일기장 (MindLog)**
   * OpenAI API 및 감정 분석 체험형 인터랙티브 위젯 탑재.
2. **오늘의 할 일 (To-Do List)**
   * 오늘의 주요 개발 및 관리 태스크를 표시하며, 완료 여부에 따라 반응하는 귀여운 AI 비서 위젯 포함.
3. **개발 마일스톤 일정표 (Dev Calendar)**
   * 이번 달 주요 스프린트 및 배포 계획을 날짜별로 확인할 수 있는 인터랙티브 달력.
4. **누적 성장 마일스톤 (Timeline)**
   * 지금까지 기획하고 배포한 프로젝트 히스토리 아카이브.
5. **AI City Builders 학습 아카이브 (🔒 패스코드: `1234`)**
   * AI 에이전트, 자동화 워크플로우 학습 영상 및 요약 문서 수록.
6. **인프라 및 구독비용 대시보드 (🔒 패스코드: `1234`)**
   * SaaS 스택 비용과 인프라 지출 명세를 투명하게 공개하는 보안 패널.
7. **생각 블로그 & 방명록**
   * 일상과 기술에 관한 단상 아카이브 및 로컬 브라우저 저장식 방명록 기능.
